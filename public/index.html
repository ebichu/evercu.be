A compact storage server — designed for 24/7-operation in your living room.

The Story

When I came across the famous Backblaze storage pod, I was immediately intrigued by its clever, no-frills design. But who really needs 67 terabytes (or rather 135 TB, with current disks) at home? I decided to "downscale" their excellent hardware design for home use, and after a long time of experimenting, the Evercube is the result.

The Evercube design is open hardware, available under a Creative Commons license. But for your convenience, I am also offering the Evercube as a Do-It-Yourself kit.

Questions & Answers

General

What can I use it for?
The Evercube is primarily intended as an "always-on" home server. 5 terabytes of raw storage capacity make it a superb file server with plenty of room for pictures, music, movies or backups. Of course it can also be used as a web server, bittorrent server, diaspora node ...
When is the kit available?
The first ten kits will be ready for delivery in November 2011.

Assembly

What is included in the kit?
The kit contains everything — minus harddisks — that is needed to build the Evercube: enclosure, motherboard (with CPU and memory), SATA backplane, disk scaffold, fan, power supply, wires, screws, nuts, washers ...
How difficult is it to put together?
Pretty easy, actually. If you know how to operate a screwdriver, you should be fine ;)
Do I have to solder anything?
No, to make this project as accessible as possible, all electrical connections are done with clamps.
What tools do I need?
Only screwdrivers and pliers.

Technical

Which disks can be used?
Any laptop (2.5" SATA) disk, including "thick" (12.5mm) 1 TB models.
Seriously — laptop disks in a server?
It might sound counterintuitive, but laptop disks actually have a lot of advantages for an always-on home server: they use less energy, run very quiet, and take up little space. Although you pay roughly double the price per gigabyte compared to 3.5" disks, you pay less for electricity, and thus save money in the long run.
What operating system does the Evercube run on?
The Sheeva Plug (which we use as motherboard) comes with Ubuntu 09.04 pre-installed on flash. You can find instructions on plugcomputer.org how to install a different operating system (including Debian, Fedora, Gentoo and FreeBSD).
How about ZFS?
Sure, ZFS would be ideal for a file server — but it is currently not available on Linux due to licensing issues. You can install FreeBSD on the Evercube, although the available memory (512 MB) might be too little for good ZFS performance.
